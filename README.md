# 🚁 Helicopter Simulation – Three.js

A fully interactive 3D helicopter simulation built with Three.js, custom shaders, and texture mapping.

## Features
- Full 3D camera control (mouse drag + WASD keys)
- Spotlight ON/OFF (left click)
- Rotor speed boost (right click)
- Wireframe mode (middle click)
- Custom GLSL shaders & texture blending
